## Getting Started

### Install the Package

Install latest version:

```bash
pip install git+https://github.com/reganto/django-ptpython
```

Install it via pip:

```bash
pip install django-ptpython
```

### Install the App

Add `django-ptpython` to your `INSTALLED_APPS` setting:

```python
INSTALLED_APPS = [
    # ...
    "django_ptpython",
    # ...
]
```
